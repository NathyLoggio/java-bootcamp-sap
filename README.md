# Java Bootcamp SAP - Continuous Learning 🚀

A repository tracking my 90-day Java bootcamp progress, focusing on core Java, data structures, and enterprise best practices. 
This project was built to consolidate my technical foundation and document my continuous learning journey.

## 🛠 Technologies
- **Language:** Java 21
- **IDE:** IntelliJ IDEA
- **Version Control:** Git & GitHub

## 🧠 My Learning Approach
Every exercise in this repository is documented with a focus on:
- **Big-O Complexity:** Time and space complexity analysis for everyday operations.
- **Memory Considerations:** Trade-offs between different data structures.
- **Enterprise Context:** Real-world SAP use cases for each concept.

## 📚 Learning Path

### Week 01: Core Concepts & Data Structures
- **`generics/`**: Implementation of custom generic classes, generic methods, and wildcard boundaries.
- **`arraylist/`**: Dynamic array operations, memory management, and internal capacity resizing.
- **`linkedlist/`**: Singly/doubly linked list manipulation, functioning as Queues and Deques.
- **`big-o/`**: Time and space complexity study notes with practical performance benchmarks.
